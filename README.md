# Linux Threat Detection Lab (SOC)
this repository demostrates detection of common attack techniques using Linux system log in a 
controlled lab environment 

##  Objetive 
To simulate real-world SOC scenarios by analizing host-based logs and identifying malicious 
activity such as brute force attacks and port scanning.

## Tools
-- Kali Linux (attacker & victim)
-- journalctl

## Skills 
-- Log analysis
-- Threat detection
-- Understanding host vs network visibility

## Note 
some attacks are not fully visible in system logs and require network-level analysis.
